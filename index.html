Media Downloader
브라우저 다운로드 + Python(yt-dlp) 명령어 생성 + 링크 BGM

0) BGM 링크 설정

mp3/ogg/wav 같은 오디오 “직접 링크”를 넣어줘. (403/CORS 차단된 링크는 재생이 안 될 수 있음)
넣어둔 링크는 저장돼서 다음에 들어와도 자동으로 재생 시도해.

예: S3/R2/본인 서버/허용된 CDN 링크
대기 중
403/CORS면 링크를 바꿔야 함
링크가 자꾸 403/CORS 나면?
그 링크 제공자가 “외부 사이트에서 직접 재생”을 막는 거야.
해결: 네가 소유한 저장소(S3/R2 등)에 올리고 CORS 허용하거나, 핫링크 허용된 CDN을 사용해야 해.

1) 브라우저로 파일 다운로드 (직접 파일 URL)

mp3/mp4/webm/wav 같은 “직접 파일 링크”를 넣으면 브라우저가 받아서 저장해줘.
유튜브/트위치 같은 서비스 URL은 보통 브라우저에서 바로 다운로드가 막혀있어(정책/CORS).

가능하면 CDN/직접 파일 주소를 넣어줘.
비워두면 URL에서 자동 추출.
대기 중

2) 파이썬(yt-dlp) 다운로드 명령어 자동 생성

아래 명령어는 네 PC에서 실행하는 방식이야. (유튜브/각종 사이트는 보통 이 방법이 안정적)
먼저: pip install -U yt-dlp

비우면 현재 폴더에 저장.
URL을 입력하면 자동으로 생성돼
