다운로드

✅ 직접 파일 URL (mp3/mp4/webm/wav 등) → 브라우저 다운로드 가능
❌ 유튜브/사이트 페이지 URL → 브라우저에서 막힘(CORS/정책) → 아래 yt-dlp 사용

URL을 입력하면 자동으로 판별해줄게.
직접 파일 URL일 때만 사용됨.
유튜브 URL은 “다운로드 시작” 대신 yt-dlp 명령어를 사용해.
